NEW BLOG

平台
vmware 17 上的ubuntu22.07
/===================================================================/
环境安装
gcc
sudo apt-get install build-essential
git
sudo apt-get install git
node.js
sudo apt install nodejs npm
检查安装
node -v
npm -v
安装hexo
sudo npm install -g hexo-cli
/===================================================================/
本地搭建博客
sudo hexo start 或 sudo hexo s
验证
用ubuntu的浏览器可以打开链接

常用命令
    hexo n "MD文章名"#新建文章
    hexo clean #清理
    hexo g #生成工程

/===================================================================/
部署博客
安装插件
sudo npm install hexo-deployer-git

git init
检查当前目录下是否有.git

sudo git config –global user.email “you@example.com“
sudo git config –global user.name “Your Name”
sudo git config –global –unset user.email “you@example.com“
sudo git config –global –unset user.name “Your Name”
