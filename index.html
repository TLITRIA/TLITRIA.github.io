NEW BLOG

平台
vmware 17 上的ubuntu22.07
/===================================================================/
环境安装
gcc
sudo apt-get install build-essential
git
sudo apt-get install git
node.js
sudo apt install nodejs npm
检查安装
node -v
npm -v
安装hexo
sudo npm install -g hexo-cli
/===================================================================/
本地搭建博客
初始化项目
sudo hexo init
运行
sudo hexo start 或简写 sudo hexo s
验证
用ubuntu的浏览器可以打开链接
常用命令
hexo n “文章名” #新建文章
hexo clean #清理
hexo g #生成工程
sudo hexo clean && sudo hexo g
/===================================================================/
部署博客
hexo安装git插件
sudo npm install hexo-deployer-git
配置./_config.yml文件
_config.yml文件只读
sudo chmod 777 ./config.yml 修改权限
deploy:
type: ‘git’
repo: https://token@github.com/用户名/用户名.github.io.git
branch: master
初始化git仓库
git init
检查当前目录下是否有.git
ls -al
配置Github用户名和注册Github用的邮箱
git config –global user.email “you@example.com“
git config –global user.name “Your Name”
删除信息
git config –global –unset user.email “you@example.com“
git config –global –unset user.name “Your Name”
在Github获取token令牌
参考https://blog.csdn.net/LanXiu/article/details/122323713
在Github创建仓库
https://blog.csdn.net/muzihuaner/article/details/113880520#
部署并访问
sudo hexo clean && hexo g
sudo hexo d
有时能提交，有时
在浏览器输入“用户名.github.io”访问
/===================================================================/
参考https://www.bilibili.com/video/BV1Yb411a7ty
